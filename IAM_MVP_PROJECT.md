# AuthMaster - IAM MVP 项目定位

> 创建日期：2026-04-02

## 项目名称：AuthMaster
- 候选：AuthMaster / IDSphere / OneAuth / TokenHub
- 定位：简化版 IDaaS / 企业级 IAM 平台

## 项目目标
- 卖给中小企业/数字化转型企业
- 作为独立产品，不依附于扫码点餐系统
- MVP 先跑通，再迭代完善

## MVP 核心功能范围

### Phase 1 MVP（必须先有）

| 模块 | 功能 | 说明 |
|------|------|------|
| **Token 管理** | JWT 签发/验证/刷新/吊销 | 核心中的核心 |
| **统一认证** | 手机号登录、邮箱登录 | 基础账户体系 |
| **第三方登录** | OAuth2.0（微信/支付宝/Google） | 可选首期 |
| **基础 RBAC** | 角色定义、权限分配、权限校验 | 最小权限控制 |
| **多租户** | 商户/企业账号隔离 | 一个企业 = 一个租户 |
| **开发者 API** | OpenAPI 供业务系统接入 | API Key 管理 |

### Phase 2（后续迭代）

| 模块 | 功能 |
|------|------|
| SSO 单点登录 | 一次登录，全局访问 |
| MFA/2FA | 多因素认证 |
| ABAC | 基于属性的动态权限 |
| 主动防御 | 防暴力破解、设备指纹 |
| 审计日志 | 登录审计、操作日志 |
| Auth SDK | 多语言 SDK |

## 技术选型（初步）

| 组件 | 技术 | 说明 |
|------|------|------|
| 后端 | Python FastAPI / Go | 高性能 API |
| 数据库 | PostgreSQL | 关系型，稳定 |
| Token 缓存 | Redis | 支持吊销、延迟低 |
| 认证协议 | OAuth2.0 / OIDC | 国际标准 |
| 前端 | Vue3 | 商户管理后台 |

## 商业模式

| 版本 | 定价 | 功能限制 |
|------|------|---------|
| 免费版 | ¥0 | 有限租户数、有限 API 调用 |
| 专业版 | ¥999/月 | 无限租户、更多 API 额度 |
| 企业版 | ¥2999/月 | SSO、MFA、专属客服 |
